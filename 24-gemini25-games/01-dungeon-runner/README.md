# Dungeon Runner

This is a simple endless runner game created with p5.js. The code is intended to be pasted into the [p5.js editor](https://editor.p5js.org/).

## Gameplay

The player chooses a character and then runs through a dungeon, jumping over and shooting obstacles. The game gets progressively faster and the goal is to get the highest score possible.

### Controls

*   **Desktop:**
    *   **Arrow Keys:** Navigate character selection.
    *   **Space/Enter:** Start the game.
    *   **Space/Up Arrow:** Jump.
    *   **F:** Shoot.
*   **Mobile:**
    *   **Tap on character:** Select character.
    *   **Tap on START:** Start the game.
    *   **Tap left side of screen:** Jump.
    *   **Tap right side of screen:** Shoot.

### Characters

There are several characters to choose from:

*   Ale (a dragon)
*   Seby (a giraffe)
*   Papino (a puffin)
*   Waffle (a waffle)
*   Silvana (a dog)

## How to Play

1.  Open the [p5.js editor](https://editor.p5js.org/).
2.  Copy the code from `public/sketch.js`.
3.  Paste the code into the editor.
4.  Press the "Play" button.
